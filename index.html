Help Yourself!

This is a group-think project. Submit updates if you have a better way of handling things, otherwise feel free to grab this code and run with it. We know how you feel -- well, hopefully you feel relief that you finally found something to help with this problem!

A New Tool For ColdFusion Programmers in the XSS War

There are few tools for ColdFusion users in the war against Cross Site Scripting attacks. This will help you. It's not perfect, it's not the only tool you'll need, but it will be a big help to you, especially if you need to pass a pen test (penetration test) of your site.

Help Wanted!

There are many improvements needed to this now-community project. Help everyone as you've now been helped, post your changes and suggestions to improve this tool!
